# rare-numbers
A tool for generating Rare Numbers

## Installation

Once you have the source for this repository checked out:

```
$ pip install -e .
```

## Usage

For more detail `rare-numbers --help`

Example:

```
$ rare-numbers --start 2 --end 9
```

## Tests

```
$ pip install -e .[test]
$ python -m pytest tests
```
